Tentang Saya

Halo, saya Della Ariska, mahasiswa Program Studi Pendidikan Matematika Universitas Muhammadiyah Kotabumi. Saya memiliki ketertarikan pada dunia pendidikan, teknologi pembelajaran, desain media interaktif, dan pengembangan website edukasi. Saat ini saya aktif mengembangkan media pembelajaran menggunakan Articulate Storyline, HTML, CSS, dan Canva AI. Saya berharap dapat menjadi pendidik sekaligus pengembang media pembelajaran yang mampu memberikan pengalaman belajar yang menarik bagi siswa.
